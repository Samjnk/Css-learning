دليل وسوم CSS الشامل

تعرف على جميع وسوم CSS ودورها في تنسيق صفحات الويب

Display

يتحكم في كيفية عرض العنصر في الصفحة

display: block; display: inline; display: flex; display: grid; display: none;
  • block: يأخذ العرض الكامل
  • inline: يأخذ عرض المحتوى فقط
  • flex: نظام تخطيط مرن
  • grid: نظام الشبكة
  • none: يخفي العنصر

Position

يحدد كيفية تموضع العنصر

position: static; position: relative; position: absolute; position: fixed; position: sticky;
  • static: الوضع الافتراضي
  • relative: نسبي لموقعه الأصلي
  • absolute: نسبي لأقرب عنصر أب
  • fixed: ثابت في الشاشة
  • sticky: يلتصق عند التمرير

Margin & Padding

التحكم في المسافات الخارجية والداخلية

margin: 10px; padding: 10px; margin: 10px 20px; padding: 10px 20px 30px 40px;
  • margin: المسافة الخارجية
  • padding: المسافة الداخلية
  • يمكن تحديد قيم مختلفة للجوانب الأربعة

Font

خصائص تنسيق النصوص

font-size: 16px; font-weight: bold; font-family: Arial; line-height: 1.5; text-align: center;
  • font-size: حجم الخط
  • font-weight: سمك الخط
  • font-family: نوع الخط
  • line-height: ارتفاع السطر
  • text-align: محاذاة النص

Background

خصائص خلفية العنصر

background-color: #fff; background-image: url('image.jpg'); background-size: cover; background-position: center; background-repeat: no-repeat;
  • background-color: لون الخلفية
  • background-image: صورة الخلفية
  • background-size: حجم الخلفية
  • background-position: موقع الخلفية
  • background-repeat: تكرار الخلفية

Border

خصائص حدود العنصر

border: 1px solid black; border-radius: 10px; border-width: 2px; border-style: dashed; border-color: red;
  • border: الحدود الكاملة
  • border-radius: تدوير الزوايا
  • border-width: سمك الحدود
  • border-style: نمط الحدود
  • border-color: لون الحدود

Flexbox

خصائص التخطيط المرن

display: flex; flex-direction: row; justify-content: center; align-items: center; flex-wrap: wrap;
  • flex-direction: اتجاه العناصر
  • justify-content: محاذاة أفقية
  • align-items: محاذاة عمودية
  • flex-wrap: التفاف العناصر

Grid

خصائص نظام الشبكة

display: grid; grid-template-columns: 1fr 1fr 1fr; grid-gap: 20px; grid-auto-rows: minmax(100px, auto);
  • grid-template-columns: أعمدة الشبكة
  • grid-gap: المسافة بين العناصر
  • grid-auto-rows: ارتفاع الصفوف

Transform

خصائص تحويل العناصر

transform: rotate(45deg); transform: scale(1.5); transform: translate(10px, 20px); transform: skew(10deg);
  • rotate: دوران العنصر
  • scale: تكبير/تصغير العنصر
  • translate: تحريك العنصر
  • skew: إمالة العنصر

Transition

خصائص الانتقالات

transition: all 0.3s ease; transition-property: transform; transition-duration: 0.5s; transition-timing-function: ease-in-out;
  • transition-property: الخاصية المتغيرة
  • transition-duration: مدة الانتقال
  • transition-timing-function: نمط الانتقال

Animation

خصائص الحركات

animation: name duration timing-function; animation-delay: 2s; animation-iteration-count: infinite; animation-direction: alternate;
  • animation-name: اسم الحركة
  • animation-duration: مدة الحركة
  • animation-timing-function: نمط الحركة
  • animation-iteration-count: عدد التكرار
